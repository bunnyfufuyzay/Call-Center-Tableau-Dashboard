# 📞 Call Center Account Phone Report Dashboard

An interactive Tableau dashboard designed to monitor call center performance across queues and time periods.

## Overview

This dashboard enables leadership teams to monitor operational performance, identify trends, and uncover opportunities for improvement.

Key capabilities include:

- KPI monitoring
- Trend analysis
- Queue ranking
- Hourly heatmap analysis
- Interactive filters
- Dynamic KPI switching
- Dynamic titles and tooltips

---

## KPIs Included

| KPI | Description | Target |
|-----|-------------|--------|
| Calls Offered | Total inbound calls received | N/A |
| Calls Handled | Total calls successfully handled | N/A |
| Service Level (SL%) | Calls answered within target | 80% |
| Abandon % | Calls abandoned before answer | < 5% |
| ASA | Average Speed of Answer | ≤ 20 sec |
| AHT | Average Handle Time | ≤ 12 min |

---

## Dashboard Features

### KPI Summary Cards
Provides overall operational metrics and performance indicators.

### Dynamic KPI Selector
Users can switch between:

- Calls Offered
- Calls Handled
- Service Level %
- Abandon %
- ASA
- AHT

The selection dynamically updates:

- Trend Chart
- Queue Ranking
- Dashboard Titles
- Tooltips

### Heatmap Analysis
Visualizes hourly performance by queue to identify bottlenecks and trends.

### Queue Ranking
Ranks queues based on the selected KPI.

## Key Insights

- Service Level exceeded the 80% target.
- HMSI handled the highest call volume.
- Queue performance remained stable across most hours.
- Lower-volume queues exhibited greater variability.
